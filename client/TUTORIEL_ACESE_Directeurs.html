TUTORIEL POUR LES DIRECTEURS D'ÉCOLE

📚 Guide d'utilisation pas à pas

Comment collecter les informations des élèves sans extrait de naissance

AVANT DE COMMENCER : Cette application fonctionne sans connexion internet ! Vous pouvez saisir les élèves hors ligne et envoyer les données plus tard quand vous avez du réseau.
1

Télécharger l'application

Accédez à l'adresse suivante sur votre téléphone ou ordinateur :

https://acesegrabo-h2jn.onrender.com/telecharger
📱

Page de téléchargement avec QR code

Cliquez sur le bouton orange :

📥 Télécharger le formulaire
Astuce : Vous pouvez aussi scanner le QR code avec votre téléphone pour accéder directement à l'application.

Le fichier "ACESE-IEPP-GRABO.html" sera téléchargé sur votre appareil.

2

Ouvrir l'application

Localisez le fichier téléchargé et ouvrez-le :

  • Sur Android : Dans "Téléchargements" ou "Downloads", tapez sur le fichier
  • Sur iPhone : Dans l'app "Fichiers" → "Téléchargements"
  • Sur PC : Double-cliquez sur le fichier dans votre dossier Téléchargements
📂

Fichier ACESE-IEPP-GRABO.html dans les téléchargements

L'application s'ouvre dans votre navigateur (Chrome, Safari, Firefox...)

Important : N'effacez pas ce fichier ! C'est votre application de travail. Conservez-le précieusement sur votre téléphone.
3

Remplir les informations de l'école

À la première ouverture, vous devez configurer votre école :

⚙️

Écran de paramétrage de l'école

Champ Que mettre ?
DRENA Sélectionnez DRENA SAN-PEDRO
IEPP Sélectionnez IEPP GRABO
Secteur pédagogique Choisissez votre secteur dans la liste (ex: SECTEUR GRABO-EST)
École Sélectionnez le nom exact de votre école (la liste apparaît après avoir choisi le secteur)
Nom du directeur Votre nom de famille
Prénoms du directeur Vos prénoms complets
Contact téléphonique 1 Votre numéro à 10 chiffres (obligatoire)
Contact téléphonique 2 Deuxième numéro si vous en avez un (facultatif)
Email Votre adresse email si vous en avez une (facultatif)

Cliquez ensuite sur :

Enregistrer & Continuer →
Astuce : Ces informations ne seront à remplir qu'une seule fois. L'application les gardera en mémoire pour la prochaine fois.
4

Saisir un élève sans extrait

Vous arrivez maintenant sur l'écran de saisie des élèves :

👨‍🎓

Formulaire de saisie d'un élève

Remplissez tous les champs obligatoires (marqués d'une *) :

Information Exemple
Nom KOUASSI
Prénoms Jean Marie
Sexe G (Garçon) ou F (Fille)
Date de naissance 15 / 03 / 2015 (jj/mm/aaaa)
Classe CP1, CP2, CE1, CE2, CM1 ou CM2
Nom du père KOUASSI Paul
Téléphone père 0123456789
Nom de la mère KOUASSI Amélie
Téléphone mère 0987654321
Nom du témoin BLÉ Koffi (voisin, oncle, etc.)
Téléphone témoin 0147258369

Cliquez sur :

➕ Ajouter l'élève
Astuce : L'application vérifie automatiquement si un élève avec le même nom et date de naissance existe déjà. Pas de doublon possible !
5

Gérer la liste des élèves

Tous les élèves saisis apparaissent dans un tableau :

📋

Liste des élèves avec options de modification

Vous pouvez :

  • ✏️ Modifier : Cliquez sur le crayon pour corriger une information
  • 🗑️ Supprimer : Cliquez sur la poubelle pour retirer un élève (avec confirmation)
  • 🔍 Rechercher : Utilisez la barre de recherche pour trouver rapidement un élève
📊 Statistiques en temps réel :
  • Total des élèves saisis
  • Nombre de garçons
  • Nombre de filles
Attention : Les données sont stockées uniquement sur votre téléphone pour l'instant. Elles ne sont pas encore envoyées à l'IEPP.
6

Envoyer les données à l'IEPP

Quand vous avez terminé la saisie et que vous avez une connexion internet (WiFi ou 4G) :

📤

Bouton d'envoi des données

Cliquez sur le bouton vert :

📤 Envoyer sur le serveur IEPP

Attendez la confirmation :

✅ "Données envoyées avec succès !"
APRÈS L'ENVOI RÉUSSI : La liste se vide automatiquement. Les élèves sont maintenant enregistrés sur le serveur de l'IEPP. Vous pouvez recommencer pour d'autres élèves si nécessaire.
Pas de connexion ? Pas de problème ! Les données restent stockées sur votre téléphone. Envoyez-les plus tard quand vous aurez du réseau. L'application fonctionne hors ligne.
7

Exporter un fichier CSV (optionnel)

Si vous souhaitez garder une copie sur votre téléphone :

Cliquez sur :

📄 Exporter CSV local

Un fichier "ACESE_[Nom de votre école]_[date].csv" sera téléchargé.

À quoi ça sert ? Ce fichier peut s'ouvrir dans Excel et contient toutes les informations des élèves. C'est une sauvegarde locale.

❓ Questions fréquentes (FAQ)

Q : J'ai effacé l'application par erreur, que faire ?

R : Retéléchargez-la à la même adresse. Vous devrez reconfigurer votre école, mais c'est rapide.

Q : Puis-je saisir sur plusieurs téléphones ?

R : Oui, chaque directeur peut installer l'application sur son propre téléphone et envoyer ses données.

Q : Que se passe-t-il si j'envoie deux fois les mêmes élèves ?

R : L'application détecte les doublons et empêche la saisie du même élève (même nom + date de naissance).

Q : Mes données sont-elles sécurisées ?

R : Oui, les données restent sur votre téléphone jusqu'à l'envoi, puis sont stockées sur un serveur sécurisé de l'IEPP.

Q : L'application consomme-t-elle du crédit téléphonique ?

R : Non pour la saisie (hors ligne). Uniquement lors de l'envoi des données (quelques Ko seulement).
